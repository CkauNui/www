Краткое описание системы

Кросс-платформенный движок для создания двумерных beat 'em up’ов, позволяющий быстро и эффективно разрабатывать игры, не прибегая к программированию. Первая версия была выпущена в 2003-м году, работала только под DOS.

Начиная с 2005-го движок переведён в статус открытого ПО (open sourse). В настоящее время поддерживается группой энтузиастов, среди которых есть несколько разработчиков оригинального Beats Of Rage (BOR). Существуют версии для таких платформ как Windows, DOS, Dreamcast, PSP, GP2X и X-Box (как бинарники, так и исполнимые файлы). Есть неофициальные версии для PS2 и GP32. Поскольку написан на C++, теоретически может быть портирован на любую платформу (при достаточных навыках кодинга).
