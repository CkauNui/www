Nintendo WiiWare

Коллекция Nintendo WiiWare (wiiware), как отдельная система для ES. (В раздаче два файла конфига для Batocera и Retrobat) 

Установка

- Формат .WAD
- Видео (480p), обложки (484x679), скриншоты, логотипы.
- gamelist.xml с русским описанием, проставлены все метаданные.
- DLC для некоторых игр.

Про настройки управления в основном посте Wii.

Краткое описание системы

WiiWare - это сервис, позволяющий владельцам приставок Wii скачивать разнообразные программы и игры за Nintendo Points (местная валюта, покупаемая за обычные деньги). В отличии от Virtual Console (ещё одного сервиса Nintendo), который предназначен для скачивания старых ретро игр, WiiWare рассчитан на малобюджетные оригинальные проекты. Уж не знаю, проблемы ли с финансированием или же просто желание порадовать фанатов стало определяющим в решении Konami выпустить на WiiWare все свои классические серии: Contra, Gradius и, конечно же, Castlevania. И чтобы подчеркнуть, что это не просто ремейки - компания добавила в название к каждому ReBirth (перерождение). Но на обычную первую Castlevania итак уже хватало разных переработок (SCIV, Chronicles). Поэтому создатели решили взять что-то не столь заезженное, например первую карманную часть серии - Adventure.

WiiWare был дополнением к Virtual Console, которая специализировалась на эмулированных играх, изначально разработанных для других систем, вместо оригинальных игр.

Установка системы в Batocera / Retrobat

Для добавления системы мы будем использовать Wii и эмулятор Dolphin, т.е. у вас в системе изначально должен поддерживаться Wii.

Retrobat:
Кинуть папки с файлами из архива в папку "\Retrobat\"
По итогу файл "es_systems_wiiware.cfg" должен лежать в "Папка с ретробатом\emulationstation\.emulationstation"

Batocera:
Кинуть папки с файлами из архива в папку "\\batocera\share\"
По итогу файл "es_systems_wiiware.cfg" должен лежать в "/userdata/system/configs/emulationstation/"
После этого wiiware игры можно кинуть в "roms\wiiware"


Установка DLC:
1. Запустить эмулятор Dolphin.
2. Перейти в "Инструменты -> Установить WAD...".
3. Установить все DLC.
