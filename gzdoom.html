DOOM (GZDOOM)

Коллекция оформлений DOOM (GZDOOM) с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

Установка системы на RetroBat

- Две версии файлов - для RetroBat и Batocera.
- Для каждого тайтла свой отдельный конфиг ".ini" (решает проблему с управлением и граф. настройками).
- Для каждого тайтла своя папка сохранений "Saves" (решает проблему общих сохранений).
- Кастомные обложки (600х900), скриншоты (360p), видео (480p), логотипы (600x300) и фан-арты.
- Проставлены все метаданные.

Краткое описание системы

GZDoom — порт движка компьютерной игры Doom, форк проекта ZDoom с обеспечением улучшенной поддержки аппаратного обеспечения (OpenGL).

- Полная поддержка 3D полов (включая наклонные и полупрозрачные)
- Отражающие горизонтальные и наклонные поверхности
- Динамическое освещение, шейдеры, светящиеся текстуры и спрайты
- Скайбоксы в стиле Quake II/Half-Life
- Качественное сглаживание графики HQNX и xBRZ для высоких разрешений
- Поддержка 3D моделей MD2 и MD3
- Полная поддержка TrueColor

ZDoom — порт, расширенная версия движка компьютерной игры Doom (Doom engine), движок позволяет играть практически во все игры на движке id Tech (Doom, Doom 2, Heretic, HeXen, Strife). Разработка движка прекращена (на версии 2.8.1), но развитие проекта продолжено как GZDoom (стабильная версия) и QZDoom (служит для испытаний новых, стабильных функций).

Рекомендации, проблемы и решения

· Изображение слишком тёмное:
Если на вашем экране изображение выглядит слишком тёмным, можно поменять "Режим освещения" и/или "Гамма-коррекция" в настройках.

Установка (для Batocera)

В Batocera эта система уже идёт по умолчанию, но для запуска нужно дополнительно кинуть "файлы и gamelist.xml" из папки "gzdoom Batocera" в "\roms\gzdoom\" и подтвердить замену.

п.с. ".cmd" файлы и папку "GZDoom" (в ней эмулятор для пк) из основной раздачи можно удалить, они нужны только для RetroBat.

Previous Next

Установка (для RetroBat)

В раздаче лежит архив с папками и файлами, которые нужно скинуть на ваш жд в папку "RetroBat". Таким образом данная система появится в списке и не затрётся в случае обновления RetroBat.

По итогу файл "es_systems_gzdoom.cfg" должен лежать в "RetroBat\emulationstation\.emulationstation\"

п.с. в папке "GZDoom" находится сам эмулятор, удалять\перемещать\переименовывать его не нужно, обновить при желании можно.
".cmd" файлы можно запускать как через RetroBat, так и просто через Windows.
