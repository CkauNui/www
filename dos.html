Краткое описание системы

Дисковая операционная система для компьютеров на базе архитектуры x86. MS-DOS — самая известная ОС среди семейства DOS-совместимых операционных систем и самая используемая среди IBM PC-совместимых компьютеров с 1980-х до середины 1990-х годов, пока её не вытеснили операционные системы с графическим пользовательским интерфейсом, в основном из семейства Microsoft Windows.

В 1981 году корпорация IBM разместила запрос на создание операционной системы, которая должна была использоваться в новом семействе компьютеров IBM PC. Microsoft выкупила права на операционную систему 86-DOS у Seattle Computer Products и начала работу по её модификации под требования IBM. Впоследствии IBM лицензировала MS-DOS и выпустила её в августе 1981 года под наименованием PC DOS 1.0. В дальнейшем разработка MS-DOS и PC DOS велась Microsoft и IBM совместно до 1993 года, когда были выпущены два раздельных релиза — 6.0 от Microsoft и 6.1 от IBM.

25 марта 2014 года музей компьютерной истории в Маунтин-Вью с разрешения компании Microsoft обнародовал исходный код операционной системы MS-DOS (версии 1.1 и 2.0), вышедшей в 1981 году, а также текстового редактора Word (1.1а) для Windows. В 2018 году был опубликован исходный код ОС MS-DOS версий 1.25 и 2.0 под свободной лицензией MIT.

Настройки эмулятора

Все файлы проверены на dosbox_pure, для нормальной работы нужно поменять настройки и сохранить их.
Или скачать прикрепленный файл dosbox.opt и поместить его в "RetroBat\emulators\retroarch\config\DOSBox-pure".

Previous Next

- В каждой папке присутствует dosbox.cfg и dosbox.conf (один для Batocera, другой для Retrobat). Эти конфиги применяют нужные настройки для конкретного тайтла. Нужно обязательно включить функцию "Loading of dosbox.conf" в настройках Retroarch.

Конвертирование аудио с дисков

Вообщем что только не пробовал, в итоге нашел один не плохой вариант. Распишу, может кому пригодится.
Нужно вытащить всё из слоя с аудио, потом сконвертировать ".wav" в ".ogg". Теперь можно удалить слой с аудио, сами образы не удаляем, ".cue" и ".bin" всё ещё нужны.
Создаём рядом с образом папку "Music" и кидаем в неё наши ".ogg".
Теперь в ".cue" нужно прописать файлы.

Отрывок из файла, на примере Hexen

Было:
FILE "Hexen.bin" BINARY
TRACK 01 MODE1/2352
INDEX 01 00:00:00
TRACK 02 AUDIO
INDEX 01 00:02:21
TRACK 03 AUDIO
INDEX 01 04:00:25
TRACK 04 AUDIO
INDEX 01 07:16:64

Стало:
FILE "Hexen.bin" BINARY
TRACK 01 MODE1/2352
INDEX 01 00:00:00
FILE "Music\Track02.ogg" MP3
TRACK 02 AUDIO
INDEX 01 00:02:21
FILE "Music\Track03.ogg" MP3
TRACK 03 AUDIO
INDEX 01 04:00:25
FILE "Music\Track04.ogg" MP3
TRACK 04 AUDIO
INDEX 01 07:16:64
