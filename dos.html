MS-DOS

Коллекция MS-DOS (dos) с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.
- В основном брал дисковые версии с музыкой/видео, а не флоппи. Аудио с дисков сконвертировано в .ogg, так игры занимают не ~700мб, а ~70мб. (Подробнее ниже)
- Все игры проверены на dosbox_pure, для нормальной работы нужно поменять настройки и сохранить их.

Scroll Lock фокусировка на игре, отключает быстрые клавиши es.
Ctrl+F9 закрыть DOSBOX (если используете standalone версию).

Настройки

- Все игры проверены на dosbox_pure, для нормальной работы нужно поменять настройки и сохранить их.
Или скачать прикрепленный файл dosbox.opt и поместить его в "RetroBat\emulators\retroarch\config\DOSBox-pure".

Previous Next

- В каждой папке с игрой присутствует dosbox.cfg и dosbox.conf (один для батоцеры, другой для ретробата). Эти конфиги применяют нужные настройки для конкретной игры. Нужно обязательно включить функцию "Loading of dosbox.conf" в настройках ретроарча.

Конвертирование аудио с дисков

Вообщем что только не пробовал, в итоге нашел один не плохой вариант. Распишу, может кому пригодится.
Нужно вытащить всё из слоя с аудио, потом сконвертировать .wav в .ogg. Теперь можно удалить слой с аудио, сами образы не удаляем, .cue .bin всё ещё нужны.
Создаём рядом с образом папку "Music" и кидаем в неё наши .ogg.
Теперь в .cue нужно прописать файлы.

Отрывок из файла, на примере Hexen

Было:
FILE "Hexen.bin" BINARY
TRACK 01 MODE1/2352
INDEX 01 00:00:00
TRACK 02 AUDIO
INDEX 01 00:02:21
TRACK 03 AUDIO
INDEX 01 04:00:25
TRACK 04 AUDIO
INDEX 01 07:16:64

Стало:
FILE "Hexen.bin" BINARY
TRACK 01 MODE1/2352
INDEX 01 00:00:00
FILE "Music\Track02.ogg" MP3
TRACK 02 AUDIO
INDEX 01 00:02:21
FILE "Music\Track03.ogg" MP3
TRACK 03 AUDIO
INDEX 01 04:00:25
FILE "Music\Track04.ogg" MP3
TRACK 04 AUDIO
INDEX 01 07:16:64
