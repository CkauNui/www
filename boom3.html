DOOM 3
(boom3 / dhewm3)

Коллекция DOOM 3 (boom3 / dhewm3) с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

Рекомендации, проблемы и решения
Установка для Batocera
Установка системы на RetroBat

- Кастомные обложки (600х900), скриншоты (480p), видео (480p), логотипы (600x300) и фан-арты.
- Рабочий мультиплеер.
- Lost Levels перекодировал звук в нужную герцовку и битрейт (иначе персонажи просто молчат) + добавлен перевод pda от ув. Rivarez.
- Адаптировал русификатор под LibreCoop и LibreCoop Resurrection of Evil (PDA назначается после загрузки карты).
- Частичная адаптация русификатора под Hard Cops.
- Для каждой игры своя папка сохранений (только RetroBat).
- Для HD-версий в папках лежат дополнительные конфиги с ультра настройками графики или настройками для слабых систем.
- Для мануалов сделал "вырезки" из журналов тех лет (5 шт).
- Проставлены все метаданные.

Для оптимизации:
Обложки, скриншоты и фан-арты в ".jpg" формате;
логотипы в ".png" обработаны через "pngquant" без потери качества.

Источники:
dhewm3 (оно же ядро boom3 в Retroarch)
Русификатор от ув. Rivarez

Краткое описание системы

Boom 3 - это порт Doom 3 и Doom 3: Resurrection Of Evil с исходным кодом GPL, известный по работе под Windows и Linux.

Doom 3 - компьютерная игра в жанре фэнтези-шутера от первого лица, разработанная компанией id Software и изданная ActiVision.

Рекомендации, проблемы и решения

Batocera - часть игр не запускается

На момент публикации (25.05.2024г) в Batocera работают не все игры\моды (пока) и выбор игры\мода осуществляется после запуска нужной игры через кнопку "Модули".

Doom 3 (папка base) - запускается по умолчанию.
Doom 3 Resurrection of Evil (папка d3xp) - после запуска выбрать модуль "d3xp".
Doom 3 HD (папка hd\base) - запускается, но текстуры отображаются неправильно.
п.с. для экономии места в Batocera остальные папки можно пока удалить.

Инвертирован курсор мыши

Зайти в быстрые настройки "Shift+F1 \ Hotkey+B > Опции ядра > Инверсия оси Y - ВЫКЛ"

COOP - как открыть PDA

Нужно назначить клавишу.
Создать сервер > Загрузиться на карту > ESC > Управление > Атака\Обзор > Coop PDA

Установка (для Batocera)

В Batocera эта система уже идёт по умолчанию, дополнительно ничего делать не нужно. Но вы можете удалить ".cmd, .dll и .exe" файлы из раздачи, они нужны только для RetroBat.

Внимание! На момент публикации (25.05.2024г) в Batocera работают не все игры\моды (пока) и выбор игры\мода осуществляется после запуска нужной игры через кнопку "Модули".

Doom 3 (папка base) - запускается по умолчанию.
Doom 3 Resurrection of Evil (папка d3xp) - после запуска выбрать модуль "d3xp".
Doom 3 HD (папка hd\base) - запускается, но текстуры отображаются не правильно.
п.с. для экономии места в Batocera остальные папки можно пока удалить.

Установка (для RetroBat)

В RetroBat данная система есть по умолчанию, использует RetroArch, но, т.к. ядро не может запустить все игры\моды, был добавлен отдельный эмулятор dhewm3 и ".cmd" файлы для запуска.

В раздаче лежит архив с папками и файлами, которые нужно скинуть на ваш жд в папку "RetroBat". Таким образом данная система появится в списке и не затрётся в случае обновления ретробата.

По итогу файл "es_systems_boom3.cfg" должен лежать в "RetroBat\emulationstation\.emulationstation\"

п.с. если вы хотите запускать игры через ядро RetroArch, то удалите файл "es_systems_boom3.cfg". (но на текущий момент ядро может запускать только основную игру и дополнение Resurrection of Evil)
