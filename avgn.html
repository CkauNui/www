The Angry Video Game Nerd

Видео-коллекция оформлений The Angry Video Game Nerd (avgn) с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

Рекомендации
Установка системы

- Запуск через ядро RetroArch "ffpmeg" (хорошо подойдёт для управления с геймпада).
- Запуск через Windows вашим плеером по умолчанию (в настройках плеера желательно выставить запуск в полноэкранном режиме).
- Просмотр видео в ES без его запуска (установить показ заставки ~ через 40m).
Специальный вид темы "Ckau-Book", по сути позволяет смотреть видео, даже если ядро на вашей системе не поддерживается.
- Сортировка видео по сезонам (включите показ папок).
- Полностью кастомное оформление с нумерацией эпизодов.
- В папке "Bonus" находятся доп. видео (Bad Game Cover Art, Board James и т.д.)

Для оптимизации:
Обложки, скриншоты и фан-арты в ".jpg" формате;
логотипы в ".png" обработаны через "pngquant" без потери качества.

Краткое описание системы

AVGN — главный герой одноимённого видеошоу серии обзоров видеоигр для игровых консолей прошлых лет. Обзоры посвящены критике низкокачественных видеоигр в юмористическом стиле.

Серия видеороликов создана Джеймсом Д. Рольфом, исполняющим роль главного героя. Шоу получило популярность благодаря публикации первых видео на сайте YouTube. После этого ролики публиковались на сайте ScrewAttack и позже, вместе с двумя другими эксклюзивными сериями, были переданы сайту GameTrailers, на котором публиковались до закрытия сайта. По данным GameTrailers, размещённые на нём серии имели от полумиллиона до двух миллионов просмотров. В последствии шоу публиковалось исключительно на YouTube-канале Cinemassacre.

На момент появления шоу называлось The Angry Nintendo Nerd. В 15-м выпуске название было изменено, чтобы не нарушать торговый знак Nintendo и в будущем начать делать обзоры игр с игровых консолей других компаний.

Рекомендации, проблемы и решения

Ядро - выбрать соотношение сторон

Для правильного отображения видео в "4:3" и "16:9" нужно в ES нажать
"Select (или Backspace) > Доп. настройки системы > Cоотношение сторон - Core Provided (определяется ядром)"

Просмотр через тему - заставка

Если будете смотреть через тему, то установите появление заставки минимум через 40 минут
"Меню > Настройки интерфейса > Настройки заставки > Заставка через - 40m"

Отображение папок

Для отображения папок (разбивки по сезонам) нужно в ES нажать
"Select (или Backspace) > Посмотреть настройки > Показать папки - Всегда"

Про управление

HotKey+B - Пауза
HotKey+A - Меню RetroArch
HotKey+Start - Выход
Left - Переход назад
Right - Переход вперёд
Up - Переход вперёд (длинный)
Down - Переход назад (длинный)
L - Смена аудио-дорожки
R - Смена субтитров

Установка

В раздаче лежит архив с папками и файлами, которые нужно скинуть на ваш жд или флешку. Таким образом данная система появится в списке и не затрётся в случае обновления ос.

Если система не отображается, переименуйте "es_systems_video_avgn.cfg" в "es_systems_avgn.cfg" (наблюдается в новых версиях ES)

RetroBat:
Кинуть папки с файлами в папку "RetroBat"
В итоге файл "es_systems_video_avgn.cfg" должен лежать в "RetroBat\emulationstation\.emulationstation\"

Batocera:
Кинуть папки с файлами в папку "\\batocera\share\"
В итоге файл "es_systems_video_avgn.cfg" должен лежать в "/userdata/system/configs/emulationstation/"
(он же "\\batocera\share\system\configs\emulationstation\")
