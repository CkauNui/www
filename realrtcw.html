Return to Castle Wolfenstein (RealRTCW)

Коллекция оформлений Return to Castle Wolfenstein (realrtcw) с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

Пока только для Retrobat. 
Для Batocera есть ioRTCW в коллекции "Ports".
п.с. ioRTCW также доступен и для Retrobat, если вы хотите ванильный опыт с небольшим количеством улучшений.

Рекомендации, проблемы и решения
Установка системы на RetroBat

- Кастомные обложки (512х512), скриншоты (480p), видео (480p), логотипы (600x300) и фан-арты.
- Русский текст от WolfETPlayer.
- Русская озвучка от Фаругус (для её отключения в папке "main" удалите файл "z_zrussiansound.pk3").
- Собраны популярные (и не очень) моды, включая неофициальные компании, которые у нас продавались на дисках.
- Вернул русскую озвучку почти во все моды (там где она была).
- Мануал, "вырезка" из журнала 2002года (1шт).
- Проставлены все метаданные. К сожалению, не смог найти точные даты выходов всех модов, поэтому некоторые даты условные (взятые из дат создания файлов).


Для оптимизации:
Обложки, скриншоты и фан-арты в ".jpg" формате;
логотипы в ".png" обработаны через "pngquant" без потери качества.

Краткое описание системы

RealRTCW - глобальный мод для классического Wolfenstein, основан на порте ioRTCW, который обеспечивает поддержку широкоформатных разрешений и 64 битных систем. Он включает в себя новый графический рендер, расширенный арсенал, переосмысленную механику стрельбы, новые высококачественные модели, текстуры и звуки.

• Расширенный арсенал, включающий в себя 12 новых оружий
• Новые HD модели для оригинального оружия
• HD текстуры для окружения и персонажей, сохраняющие стиль и атмосферу оригинала
• Поддержка погодных эффектов - дождь и снег
• Новые предметы для инвентаря - адреналин и бинты
• Система реальной отдачи
• Обновлённые характеристики оружия
• Улучшенный ИИ
• Новые звуки высокого качества
• Новый интерфейс
• Новая система уровней сложности
• Поддержка субтитров
• Интеграция с Steam Workshop для пользовательских карт и модификаций
• Поддержка достижений Steam
• Восстановленный контент включающий в себя тренировочный уровень и неиспользованный саундтрек

Дополнительная информация (больше для себя)

При записи видео\скриншотов была такая же проблема, как и у tyrquake - "картинка выглядит намного темнее, чем есть на самом деле, т.е. если выкрутить в яркость на максимум, то на моём экране всё будет пересвечено, но если сделать скриншот\видео, то яркость там будет на минимум"
Решить проблему удалось с помощью записи через Fraps.

Установка (для RetroBat)

В раздаче лежит архив с папками и файлами, которые нужно скинуть на ваш жд в папку "RetroBat". Таким образом данная система появится в списке и не затрётся в случае обновления RetroBat.

По итогу файл "es_systems_realrtcw.cfg" должен лежать в "RetroBat\emulationstation\.emulationstation\"

п.с. ".cmd" файлы можно запускать как через RetroBat, так и просто через Windows.
