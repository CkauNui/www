Hypseus Singe

Коллекция оформлений Hypseus Singe (singe), с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

Это логическое развитие эмулятора Daphne.

Установка на Batocera и Retrobat

Краткое описание системы

Hypseus SINGE, Somewhat Interactive Nostalgic Game Engine, (названный в честь дракона из Dragon's Lair) - это логическое развитие эмулятора Daphne, в котором используется скриптовая система на основе Lua, позволяющая быстро симулировать тайтлы с LaserDisc или создавать совершенно новые тайтлы, на основе видеофайлов в HD качестве! Язык прост в освоении, очень мощный и быстрый. Все функции, необходимые для разработки собственного тайтла, доступны через простой интерфейс прикладного программирования (API).

Установка (Batocera, RetroBat)

На текущий момент, в Batocera (v41) тайтлы для эмулятора Singe должны находиться в папках с названием .daphne на конце, а в Retrobat (v.7.3.2.1) — с названием .hypseus.

Рядом с файлом gamelist.xml в папке roms\singe находятся два .bat-файла для автоматического переименования папок. Запустите тот, который соответствует вашей системе (_Batocera - daphne.bat или _Retrobat - hypseus.bat). Также вы можете переименовать папки вручную.

Пути к папкам с обоими названиями уже прописаны в gamelist.xml, поэтому править его не нужно.

Рекомендации, проблемы и решения

Если изображение не вмещается в экран

Select > Доп. настройки системы > Видео > Game Window Downscale - подобрать под свой экран (у меня это 80-85%)

Для тайтлов с припиской [16:9] нужно поменять настройки

Для широкоформатных тайтлов добавлена приписка [16:9] в названии. Для них нужно поменять настройки в EmulationStation (ES).
Удерживать кнопку запуска > Доп. настройки > Соотношение сторон - 16/9
