Commodore 64 Games System

Коллекция оформлений Commodore 64 Games System (c64gs), с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

Сюда вошли тайтлы, официально вышедшие на платформе C64GS, а также те, которые планировались к выходу. Это далеко не все тайтлы для C64, поддерживающие геймпад — остальные будут в коллекции C64 (скорее всего).

Забавный факт: люди, официально купившие Terminator 2 в своё время, не могли её запустить. На титульном экране требовалось нажать кнопку на клавиатуре, которой у модели GS не было…

Краткое описание системы

Commodore 64 Games System (сокр. C64GS) — игровая приставка с картриджами, созданная компанией Commodore на базе популярного домашнего компьютера Commodore 64. Устройство было выпущено в декабре 1990 года, в попытке войти на бурно развивающийся рынок игровых устройств. Продажи проходили только в Европе — в Великобритании и Германии. В целом этот проект признаётся коммерческим провалом: было произведено около 80 тысяч машин, лишь 25% от этого числа было продано, остальные были возвращены и использованы для выпуска компьютеров C64.

C64GS была не первой попыткой Commodore сделать приставку из C64: в 1982 году была выпущена Commodore MAX Machine — сильно урезанный вариант компьютера, ориентированный только на игры. C64GS была гораздо ближе к C64 и совместима с ним.

Установка (для Batocera)

В раздаче лежит архив с папками и файлами, которые нужно скинуть на вашу флешку\жд по пути "\\batocera\share". Таким образом данная система появится и не затрётся в случае обновления Batocera.

По итогу файл "es_systems_c64gs.cfg" должен лежать в "/userdata/system/configs/emulationstation/"
(он же "\\batocera\share\system\configs\emulationstation\")

Установка (для RetroBat)

В раздаче лежит архив с папками и файлами, которые нужно скинуть на ваш жд в папку "RetroBat". Таким образом данная система появится в списке и не затрётся в случае обновления RetroBat.

По итогу файл "es_systems_c64gs.cfg" должен лежать в "RetroBat\emulationstation\.emulationstation\"

Настройки, проблемы и решения

· Выставить модель устройства:
Для всей системы можно выставить GS модель (кроме Terminator 2).
[RetroBat]: Select > Доп. настройки системы > Emulation > Model Type - C64 GS PAL
[Batocera]: Select > Доп. настройки системы > Тип модели - C64 Games System PAL

Previous Next

· Вторая кнопка:
В большинстве тайтлов для "действия" (стрельбы и т.д.) использовалась только одна кнопка, вторая обычно не была задействована. Однако в настройках EmulationStation: Select > Доп. настройки системы > Controls > Retropad Face Button Options можно настроить различные комбинации кнопок — например, назначить одну для стрельбы, а вторую для прыжка (при этом кнопка "UP" дублирует нажатие "Вверх" на джойстике).

Terminator 2 не нажимаются кнопки на титульном экране

Эта проблема постигла людей, официально купивших картридж в своё время. На титульном экране требовалось нажать кнопку на клавиатуре, которой у модели GS не было…

[RetroBat]: Удерживать кнопку запуска > Доп. настройки > Emulation > Model Type - C64 PAL
[Batocera]: Удерживать кнопку запуска > Доп. настройки > Тип модели - C64 PAL

Запустить тайтл, на титульном экране вызвать клавиатуру через "Select" и нажать на "T".

В Fun Play и в Pang не работает управление

В большинстве тайтлов джойстик используется во 2-м порту, но для Fun Play и Pang его нужно переключить в 1-й.

[RetroBat]: Удерживать кнопку запуска > Доп. настройки > Controls > Joystick Port - PORT 1
[Batocera]: Удерживать кнопку запуска > Доп. настройки > Порт контроллера - ПОРТ 1
