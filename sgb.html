Super Game Boy

Коллекция оформлений Super Game Boy (sgb) с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

SGB адаптер позволяет запускать картриджи для портативной консоли Game Boy и Game Boy Color через приставку SNES/Super Famicom. Тайтлы для Game Boy, вышедшие после Super Game Boy, были оптимизированы для запуска с ним и содержали в своём коде добавочную информацию, позволявшую менять цвета на экране, отображать специальную рамку (можно включать\отключать по желанию) и даже воспроизводить скрытые ранее спрайты, эффекты, дополнительные звуки и подключать второй геймпад для двух пользователей.

- Обложки (~700x700), видео (480p).
- Фан-арты (~125 из 180).
- Проставлены все метаданные.
- Видео, скриншоты (с рамками, если они есть) с правильным соотношением сторон.

Для оптимизации:
Обложки, скриншоты и фан-арты в ".jpg" формате;
логотипы в ".png" обработаны через "pngquant" без потери качества.

Краткое описание системы

Super Game Boy является 16-битным картриджем-адаптером для игровой консоли Super Nintendo Entertainment System, в Японии также известной как Super Famicom. Super Game Boy позволяет запускать картриджи с играми для портативной консоли Game Boy на экране телевизора, используя в качестве игрового контроллера геймпад от SNES/Super Famicom. На момент выпуска в США Super Game Boy стоил $59.99. В Великобритании он продавался по цене £49.99. Super Game Boy является предшественником устройства Game Boy Player для консоли Nintendo GameCube, которое работало по схожей схеме.

Super Game Boy был совместим с картриджами для оригинального монохромного Game Boy и картриджами для Game Boy Color, если они поддерживали игру в чёрно-белом режиме. Он мог отображать только четыре оттенка зелёного цвета. Игры для Game Boy, вышедшие после Super Game Boy, были оптимизированы для игры с ним и содержали в своём коде добавочную информацию, позволявшую менять цвета на экране, отображать на экране телевизора специальную рамку и даже воспроизводить скрытые ранее спрайты, которые были, например, на загрузочном экране Mario's Picross. Такие игры содержали специальный логотип «Super Game Boy Game Pak» на коробке и на самом картридже. Всего этот адаптер поддерживал 64 цвета рамки и 12 цветов экрана. Статические изображения Super Game Boy мог воспроизводить в 10 цветах.
