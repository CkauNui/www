Game Boy + Color "ТОП"

Коллекция оформлений Game Boy + Game Boy Color "ТОП" (gb_top) с русским описанием, обложками, логотипами, видео, скриншотами и фан-артами.

Хоть для каждого "ТОП" свой, постарался собрать некую "выжимку".

Установка системы на Batocera
Установка системы на RetroBat

- Кастомные обложки (600х900). Обычные обложки (~600х600) лежат в отдельной папке, т.е. на выбор у вас два вида обложек.
- Кастомные фан-арты.
- Проставлены все метаданные.
- Видео, скриншоты с правильным соотношением сторон.
- Манулы для серии Legend of Zelda (сделал вырезки из энциклопедий)

Краткое описание Game Boy

Линейка портативных игровых устройств, разработанная и производившаяся Nintendo.
Первый Game Boy был выпущен ещё в 1989 году, как продолжение к известным карманным играм Nintendo Game & Watch. Это была первая популярная портативная игровая система, и многие компании пытались повторить успех Game Boy, но никому сделать этого так и не удалось.

Устройство имело монохромный жидкокристаллический экран с размерами 160×144 пикселей, стереозвук на четыре канала и функцию GameLink, позволяющую соединять две консоли вместе и играть вдвоём в поддерживающие данную технологию игры.

Всякий раз, если по каким-либо причинам продажи Game Boy шли на убыль, Nintendo выпускала новые разновидности Game Boy, чтобы снова увеличить продажи. Таким образом, в ноябре 1996 года появился Game Boy Pocket, который был меньше в размерах на 30 %, чем свой предшественник, но с экраном большего размера и всего с двумя батареями AAA. Позже был выпущен Game Boy Color c цветным экраном, затем — Game Boy Advance.

Краткое описание Game Boy Color

Портативная игровая система, наследница Game Boy, выпущенная компанией Nintendo 21 октября 1998 года в Японии, в ноябре 1998 года в Северной Америке и в 1999 году в Европе.

Игровая консоль оснащалась ЖК-дисплеем с цветовой гаммой, сравнимой с NES. Для системы вышло более 700 игр в различных жанрах (аркады, RPG, стратегии, симуляторы, спортивные игры), большую долю игр составляли игры про вселенную покемонов (Pokemon). Также выходили игры в самых неожиданных для портативной консоли жанрах, например, в жанре Survival Horror (Alone in the Dark: The New Nightmare и Resident Evil Gaiden).

Спад продаж Game Boy Color наступил в начале 2001 года, когда Nintendo выпустила новую версию Game Boy — Game Boy Advance.

Рекомендации, проблемы и решения

· Если не запускается:
Принудительно выставить эмулятор (не "АВТО"). Для этого заходим в нужную систему, нажимаем "Select" (backspace) > Дополнительные настройки системы > Эмулятор > выставляем Libretro:Gambatte" или любой другой.

Установка (для Batocera)

В раздаче лежит архив с папками и файлами, которые нужно скинуть на вашу флешку\жд по пути "\\batocera\share". Таким образом данная система появится и не затрётся в случае обновления Batocera.

По итогу файл "es_systems_gb_top.cfg" должен лежать в "/userdata/system/configs/emulationstation/"
(он же "\\batocera\share\system\configs\emulationstation\")

п.с. Если вы не хотите устанавливать отдельную систему, то можете переименовать папку с коллекцией "gb_top" в "gbc" и кинуть её в roms.

Установка (для RetroBat)

В раздаче лежит архив с папками и файлами, которые нужно скинуть на ваш жд по пути в папку "RetroBat". Таким образом данная система появится и не затрётся в случае обновления RetroBat.

По итогу файл "es_systems_gb_top.cfg" должен лежать в "RetroBat\emulationstation\.emulationstation\"

п.с. если вы не хотите устанавливать отдельную систему, то можете переименовать папку с коллекцией "gb_top" в "gbc" и кинуть её в roms.
